# Dotfiles

This repository contains my personal dotfiles for a _Gentoo_ Linux setup, tailored for a
Wayland-based environment.
It includes configurations for my window manager, terminal, shell, editor, and more.

## System Overview 

- **Distribution**: _Gentoo Linux_
- **Display Protocol**: Wayland
- **Window Manager**: [Hyprland](https://github.com/hyprwm/Hyprland)
- **Terminal**: [Kitty](https://sw.kovidgoyal.net/kitty/)
- **Shell**: [Zsh](https://www.zsh.org)
- **Panel**: [Waybar](https://aur.archlinux.org/packages/waybar-hyprland-git)
- **Notification Daemon**: [Mako](https://github.com/emersion/mako)
- **Launcher**: [Wofi](https://hg.sr.ht/~scoopta/wofi)
- **File Manager**: [Yazi](https://yazi-rs.github.io/)
- **Editor**: [Neovim](https://neovim.io/)

## Installation

I use [yadm](https://yadm.io/) to manager this repo, which is a lightweight tool for
managing your dotfiles using Git.
